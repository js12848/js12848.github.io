What the assignment was

Choose a Paul Rand layout and recreate it using HTML and CSS. Focus on type hierarchy, alignment, and white space rather than inventing a new design.

Challenges

  • Matching the exact spacing and rhythm between text blocks.
  • Deciding when to use flexbox vs fixed widths vs margins.
  • Typographic details: line-height, font-weight, and making the page feel light, not cramped.
  • hierarchy how to arrange them.

What I learned

  • Copying is a way of reading in design learning how these artistic page works
  • More css and how css can be very interesting.
