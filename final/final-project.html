Habit Mirror

A page that reflects how we already use the web

This project is a single-page experience that mirrors the small habits we carry online: scrolling too fast, hovering for cues, and reloading without thinking.

Ideas

The page begins clean and minimal. As the user interacts, subtle responses appear: scroll speed affects clarity, notification react to hovering, and small messages respond to reloading. Nothing is saved or personalized, but the site behaves like it notices what you do.

Why this project

After watching The Social Dilemma and The Antisocial Network, I revisited my own projects and realized how automatic my online behaviors have become. This made me question what I now overlook, expect, or perform without thinking. Habit Mirror is a way to expose those patterns through simple, readable interactions.

Key interactions

  • Scroll-based effects: scrolling too fast causes text to blur or fade, showing how speed changes what we actually see.
  • Hover-triggered “notifications” that briefly appear, referencing how platforms pull our attention with tiny signals.
  • A subtle reaction when the user refreshes the page, highlighting the habit.

How I’ll build it

  • HTML and a simple, intentional layout.
  • Lightweight JavaScript that listens for scroll, hover, and refresh behaviors.
  • A short explanation tying the interactions back to platform habits and attention.
